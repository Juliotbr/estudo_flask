# Meu Bloco de Notas

Siga os passos abaixo para rodar o aplicativo diretamente no seu computador usando o VS Code.

## 1. O que você precisa ter instalado
- **Python**
- **Visual Studio Code (VS Code)** com a extensão oficial do **Python** instalada.

## 2. Instalar as dependências
Abra o terminal do próprio VS Code (menu `Terminal` > `New Terminal`) e rode o seguinte comando para instalar o que o projeto precisa:

```bash
pip install Flask Flask-SQLAlchemy
```

## 3. Como abrir o aplicativo
1. Abra a pasta completa do projeto no seu **VS Code**.
2. No painel do lado esquerdo, clique no arquivo `app.py` para deixá-lo aberto na sua tela.
3. Olhe para o canto superior direito do VS Code e clique no **botão de Play** (o ícone de triângulo que diz *Run Python File*).
4. O terminal vai abrir na parte de baixo da tela mostrando algumas mensagens de que o servidor iniciou.

## 4. Acessar no navegador
Com o botão Play rodando, abra o seu navegador de internet (Chrome, Edge, etc.) e acesse este endereço:

👉 **http://127.0.0.1:5000**

Pronto! O sistema vai abrir e o banco de dados será criado sozinho na primeira vez que você acessar.
